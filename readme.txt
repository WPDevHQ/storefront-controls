Re-order or disable any of the header/footer components in Storefront theme.

== Description ==

Using Storefront Controls, any component that is hooke in to the storefront_header or storefront_footer can be re-order or disable, and control the display and order of any function hooked in to these hook.

Looking for a helping hand? [Get In Touch At](http://wpdevhq.com/).

== Usage ==

Once activated, 2 items namely "Header Control" and "Footer Control" will display in the Theme Customizer ( Appearance > Customizer ).

Visiting either screen will display a table of the possible components with options for re-ordering them, or disabling individual components.

Once done, click the "Save Changes" button, to commit your new component order into the system.

== Installation ==

Installing "Storefront Controls" can be done either by searching for "Storefront Controls" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org.
1. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the "Appearance > Customizer" section, select "Homepage Control". There you can select the components you'd like to display, and in which order.

== Frequently Asked Questions ==

= The sections listed above do not appear in the Customizer. Is my theme broken? =

Not at all. This plugin is design to sorely work with Storefront theme.

Please download or search and install the theme to take advantage of this plugin's features.

= What happens if I switch themes? Do I lose my settings? =

Not at all. :) If you switch themes, Storefront Controls will remember the header/footer component settings you applied making switching themes a breeze.
